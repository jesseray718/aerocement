## 3. Thermodynamic Modeling

### 3.1 Solar Chimney (Power Generation)

Natural convection (stack effect) drives a Stirling engine (*future phase*).

Rayleigh Number:

    Ra = (g * beta * Delta_T * L^3) / (nu * alpha)

- g = 9.81 m/s^2
- beta ~ 1/300 K^-1 (air)
- Delta_T = 80 C (24 ft prototype) / 176 C (100 ft theoretical)
- L = 7.32 m (24 ft) / 30.5 m (100 ft)
- nu = 1.5e-5 m^2/s
- alpha = 2.2e-5 m^2/s

Result: Ra >> 10^7 -- turbulent, supercritical (*modeled*).

Power Output:

    P = eta_Stirling * alpha_absorb * I * A

- eta_Stirling ~ 10-20% of Carnot (*theoretical*)
- alpha_absorb = 0.98 (*pending*)
- I = 800 W/m^2
- A = collector area (variable)

Estimated: 0.5-2.0 kW (24 ft) / ~15.7 kW (100 ft + 100 m^2) -- *pending validation*.

### 3.2 Geothermal Evaporative Cooling

Three stages:

1. **Geothermal pre-cooling**: Air stabilizes at 55 F (13 C) at 10 ft depth.
2. **Desiccant dehumidification**: Waste heat dries air to <1% RH (*theoretical*).
3. **Evaporative cooling**: Dry air flows through wet aerocement, targeting 35 F (1.7 C) (*pending*).

The underground labyrinth is where S/V advantage is critical. Air flows *through* porous walls at geothermal temperature, not just past a pipe wall.

Wet-Bulb Limit: For 55 F air at near-zero RH, adiabatic saturation approaches 35 F (1.7 C) (*modeled*).

Cooling Power:

    Q_cool = m_dot_air * cp * Delta_T

- m_dot_air = 0.56 kg/s (1,000 CFM, *estimated*)
- cp = 1.005 kJ/kg-K
- Delta_T = 20 F (11.3 C)

Estimated: ~6.4 kW (*pending validation*).

### 3.3 Desiccant Dehumidification

Silica gel or zeolite removes moisture, regenerated by solar chimney waste heat.

    m_dot_water = Q_cool / L_v

- L_v = 2,260 kJ/kg (latent heat of vaporization)

Estimated: ~2.8 kg/h (*pending*).

#!/usr/bin/env python3
"""
Aerocement Prototype V1 Data Logger
Termux-compatible. Logs temperature and humidity to CSV.

Replace stub functions with real sensor reads (DS18B20, DHT22, etc).
"""

import time
import csv
import os
from datetime import datetime

LOG_DIR = os.path.expanduser("~/aerocement/data")
LOG_FILE = os.path.join(LOG_DIR, "log.csv")

SENSORS = {
    "T1": "Tunnel intake",
    "T2": "After desiccant",
    "T3": "Tunnel outlet",
    "T4": "Cooler interior",
    "T5": "Solar outlet",
}

HYGROMETERS = {
    "H1": "Before desiccant",
    "H2": "After desiccant",
}


def read_temp(sensor_id):
    """Replace with real DS18B20 read."""
    return 25.0 + (time.time() % 10)  # SIMULATED


def read_humidity(sensor_id):
    """Replace with real DHT22 read."""
    return 50.0 + (time.time() % 20)  # SIMULATED


def main():
    os.makedirs(LOG_DIR, exist_ok=True)
    header = ["timestamp"] + list(SENSORS.keys()) + list(HYGROMETERS.keys())

    print(f"Logging to {LOG_FILE}")
    print(f"Sensors: {header}")

    with open(LOG_FILE, "a", newline="") as f:
        writer = csv.writer(f)
        if os.path.getsize(LOG_FILE) == 0:
            writer.writerow(header)

        while True:
            row = [datetime.now().isoformat()]
            row += [read_temp(sid) for sid in SENSORS]
            row += [read_humidity(hid) for hid in HYGROMETERS]
            writer.writerow(row)
            f.flush()
            print(f"  {row[0]} | T1={row[1]:.1f} T2={row[2]:.1f} T3={row[3]:.1f} T4={row[4]:.1f} T5={row[5]:.1f} H1={row[6]:.1f} H2={row[7]:.1f}")
            time.sleep(60)


if __name__ == "__main__":
    main()
